## Description

The `utils.py` file provides a collection of utility functions and helpers used throughout the `cache_manager` package. These utilities support common operations such as time parsing and formatting, type checking, serialization, deserialization, and other general-purpose tasks that facilitate cache management and data handling. By centralizing these reusable functions, the module promotes code reuse, consistency, and maintainability across the package.
